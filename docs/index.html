Following task is to calculate Bayes posterior mean, so we need to have an initial distribution. As depicted in books example 2.7, one way to construct an initial distribution is [METHODOLOGY]

We have following equations that parameters must satisfy in order to:

\[\begin{align*} E(nvl_j) &= \frac{n_j\alpha}{\alpha + \beta} \\ Var(nvl_j) &= \frac{n_j\alpha\beta(\alpha + \beta + n_j)}{ (\alpha + \beta)^2(\alpha + \beta+1)} \end{align*}\]

Parameters of the previous system of equations are:

  • \(nvl_j\) as number of survey respondents with a very liberal stance in state \(j\).
  • \(n_j\) number of surveys taken on \(j\) state.

Rewriting the system in terms of proportions, it results:

\[\begin{align*} E\left(\frac{nvl_j}{n_j}\right) &= \frac{\alpha}{\alpha + \beta} \\ Var\left(\frac{nvl_j}{n_j}\right) &= \frac{\alpha\beta(\alpha + \beta + n_j)}{n_j(\alpha + \beta)^2(\alpha + \beta+1)} \end{align*}\]

In this case, we should note that to construct proportion in the variance equation, we should divide by constant \(n_j^2\) both sides, while \(nvl_j\) remains as unknown parameter. Then we replace expected value and variance with equivalent sample values. Similarly, \(n_j\) is replaced with sample mean (number of surveys by state in average). # Creamos la funcin de verosimilitud
crear_log_verosim <- function(n_very_liberal, n, ...){

  log_verosimilitud_pct <- function(pct){
    log_prob <- n_very_liberal * log(pct) + (n - n_very_liberal) * log(1 - pct)
    log_prob
  }
  log_verosimilitud_pct
}

# Calculamos el estimador de mxima verosimilitud de cada estado
conteo_edo <- conteo_edo %>% 
  mutate(ml_estim=map_dbl(
          pmap(., crear_log_verosim), 
          ~optimize(.x, c(0, 1), maximum = TRUE)[['maximum']]
          )
        ) %>%
  arrange(ml_estim)

Solucin

Primero construyamos una distribucin inicial de los datos. EL mtodo ser igual al utilizado en el ejemplo 2.7 citado en el planteamiento del ejercicio. En este caso tenemos lo siguiente:

\[\begin{align*} E(nvl_j) &= \frac{n_j\alpha}{\alpha + \beta} \\ Var(nvl_j) &= \frac{n_j\alpha\beta(\alpha + \beta + n_j)}{ (\alpha + \beta)^2(\alpha + \beta+1)} \end{align*}\]

En las frmulas anteriores, \(nvl_j\) denota el nmero de personas encuestadas que manifestaron ser de una posicin very liberal en el estado \(j\), mientras que \(n_j\) denota el nmero de encuestas realizadas en el mismo estado. Como estamos modelando en trminos de proporciones, las ecuaciones anteriores quedan como sigue:

\[\begin{align*} E\left(\frac{nvl_j}{n_j}\right) &= \frac{\alpha}{\alpha + \beta} \\ Var\left(\frac{nvl_j}{n_j}\right) &= \frac{\alpha\beta(\alpha + \beta + n_j)}{n_j(\alpha + \beta)^2(\alpha + \beta+1)} \end{align*}\]

Notemos que en el caso de la varianza, para construir la proporcin dividimos ambos lados entre \(n_j^2\) (en este caso como una constante), pues \(nvl_j\) es el parmetro de la distribucin que buscamos. El valor esperado y la varianza los reemplazaremos con los valores muestrales. De igual manera, el valor de \(n_j\) a la derecha lo reemplazaremos con la media muestral, es decir, el nmero de encuestas promedio que se realizaron en cada estado. Calculemos con R dichos parmetros:

De lo anterior, el sistema a resolver queda como sigue: \[\begin{align*} 0.04645082 &= \frac{\alpha}{\alpha + \beta} \\ 0.0003306732 &= \frac{\alpha\times\beta(\alpha + \beta + 640.2292)}{640.2292(\alpha + \beta)^2(\alpha + \beta+1)} \end{align*}\]

Resolviendo el sistema en Wolphram Alpha obtenemos el siguiente resultado:

Ecuaciones insertadas en https://www.wolframalpha.com/calculators/system-equation-calculator
0.04645082 = (a) / (a + b)
0.0003306732 = (a b * (a + b + 640.2292))/(640.2292 (a + b)^2*( a + b + 1 ) )
a >=0
b >=0

\[\begin{align*} \alpha &= 7.80945 \\ \beta &= 160.314 \end{align*}\]

Con lo anterior, ya tenemos los valores para \(\alpha\) y \(\beta\) de la distribucin inicial \(Beta\). Ahora calcularemos el ajuste de cada estado. Para esto, usaremos resultados de anlisis conjugado Beta-Binomial, que nos dicen que si nuestro modelo ser \(nvl_j \sim Binomial (n_j, p)\), con una inicial \(Beta(\alpha, \beta)\) para \(p\), entonces la posterior tendr una distribucin \(Beta(nvl_j+\alpha, n_j-nvl_j+\beta)\). El estimador que usaremos ser la media posterior de cada estado.

gen_media_post <- function(n_very_liberal, n, ...){
  mean(rbeta(50000, n_very_liberal + alpha,
             n - n_very_liberal + beta))
}
